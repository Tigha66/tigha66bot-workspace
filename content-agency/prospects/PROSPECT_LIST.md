## Priority Scoring System

Score each prospect 1-10:

**Budget (0-3 points)**
- 3 = Recently funded / Enterprise / High-ticket
- 2 = Growing company / Clear budget signals
- 1 = Small but viable / Unclear budget
- 0 = No budget signals / Too small

**Need (0-3 points)**
- 3 = Clear pain point (no content, inconsistent, etc.)
- 2 = Some content but not enough
- 1 = Decent content but could improve
- 0 = Already doing content well

**Authority (0-2 points)**
- 2 = Decision maker (Founder, CMO, VP)
- 1 = Influencer (Director, Manager)
- 0 = No decision power

**Accessibility (0-2 points)**
- 2 = Easy to reach (responsive, connected)
- 1 = Somewhat accessible
- 0 = Hard to reach / Gatekept

**Total Score:**
- 8-10 = Tier 1 (Contact immediately)
- 5-7 = Tier 2 (Contact this week)
- 3-4 = Tier 3 (Contact next week)
- 0-2 = Tier 4 (Nurture / deprioritize)

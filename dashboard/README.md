# 📊 LeadFlow AI Business Dashboard

A beautiful, real-time dashboard to track all your projects, websites, and revenue streams in one place.

---

## 🚀 **Features**

- **Revenue Tracking** - Total pipeline value across all projects
- **Project Overview** - Status and metrics for all active projects
- **Website Portfolio** - View all 69 deployed websites
- **Quick Actions** - High-priority tasks to focus on
- **Real-time Stats** - Live metrics and KPIs
- **Modern UI** - Beautiful glassmorphism design with animations

---

## 📁 **What's Tracked**

### **Projects:**
1. **LeadFlow AI** - Email outreach automation (200/day)
2. **Website Builder Service** - 69 UK business websites
3. **Agency Program** - White-label resell program
4. **Auto Lead Gen** - Tavily + Firecrawl automation

### **Metrics:**
- Emails sent, responses, demos, closed deals
- Websites deployed, niches covered, cities
- Revenue projections and timelines
- Quick action items with priorities

## 🛠️ **Setup**

### **Install Dependencies:**
```bash
npm install
```

### **Start Dashboard:**
```bash
npm start
```

### **Access:**
Open http://localhost:3000 in your browser

---

## 📊 **API Endpoints**

- `GET /api/projects` - All project data
- `GET /api/websites` - Deployed websites list
- `GET /api/stats` - Revenue statistics
- `GET /` - Dashboard UI

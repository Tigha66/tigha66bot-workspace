⚡ Key API Endpoints

POST /api/adapters

Register agents, send heartbeats

POST /api/agents/register

Self-register new agents

GET /api/agents

List all registered agents

GET /api/tasks

List all tasks

GET /api/sessions

List all sessions

POST /api/auth/login

User authentication

🚀 Quick Start

1. Register Your Agent:

curl -X POST http://localhost:3000/api/adapters \
-H "Content-Type: application/json" \
-H "x-api-key: YOUR_API_KEY" \
-d '{
"framework": "generic",
"action": "register",
"payload": { "agentId": "tigha66bot", "name": "Tigha66bot" }
}'

2. Send Heartbeat (every 5 min):

curl -X POST http://localhost:3000/api/adapters \
-H "Content-Type: application/json" \
-H "x-api-key: YOUR_API_KEY" \
-d '{
"framework": "generic",
"action": "heartbeat",
"payload": { "agentId": "tigha66bot", "status": "online" }
}'
